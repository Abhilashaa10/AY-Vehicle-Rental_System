# Vehicle Class - Java OOP

##  Overview

A simple Java program demonstrating the basics of **Object-Oriented Programming (OOP)** by creating a `Vehicle` class and multiple objects.

##  Concepts Practiced

* Class
* Object
* Instance Variables
* Object Creation (`new`)
* Accessing Object Properties

##  Output

```text
Rent of Toyota with plate number MH04HX1234 is 3500.0
Rent of Hero Honda with plate number MH12UI9879 is 560.89
```

This project is part of my Java OOP learning journey, where I'm practicing one concept at a time and uploading it to GitHub.
